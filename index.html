¿Qué es una integral?

Una integral sirve para sumar cantidades muy pequeñas. Se usa para calcular el área bajo una curva y otras acumulaciones.

Tipos de integrales

  • Indefinida: da una familia de funciones.
    ∫ f(x) dx = F(x) + C
  • Definida: da un número entre a y b.
    ∫ab f(x) dx = F(b) − F(a)

Reglas básicas

  • Potencia: ∫ xn dx = xn+1/(n+1) + C (si n ≠ −1)
  • Constante: ∫ k dx = kx + C
  • 1/x: ∫ (1/x) dx = ln|x| + C

Ejemplos

Ejemplo 1 (indefinida):

∫ 3x2 dx = x3 + C

Ejemplo 2 (indefinida):

∫ (2x + 5) dx = x2 + 5x + C

Ejemplo 3 (definida):

∫02 x dx = (22/2) − (02/2) = 2

✅ Conclusión: la integral ayuda a calcular acumulaciones como el área bajo la curva.
